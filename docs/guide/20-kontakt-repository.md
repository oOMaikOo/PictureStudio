# 🔗 Kontakt & Repository

PictureStudio ist Open-Source. Fragen, Fehlerberichte und Pull-Requests sind willkommen.

## GitHub-Repository

[https://github.com/oOMaikOo/PictureStudio](https://github.com/oOMaikOo/PictureStudio)

  - **Issues** — Fehler melden oder Feature-Wünsche eintragen
  - **Releases** — fertige Versionen als ZIP herunterladen

## Feedback & Beiträge

Pull Requests sind willkommen. Bitte einen Feature-Branch erstellen und
einen kurzen Issue anlegen, bevor größere Änderungen umgesetzt werden.
