# 🎬 Video-Annotation

Direkte Annotation einzelner Video-Frames ohne vorherigen Frame-Export.

## Video laden

Klicke **Video laden…** und wähle eine Videodatei (MP4, AVI, MOV, MKV …).

## Frame-Navigation

Schieberegler verschieben → Frame-Nummer und Zeitstempel werden angezeigt.

## Frame extrahieren & labeln

**Frame extrahieren** → Label aus Dropdown wählen → **Zum Projekt hinzufügen**.

> 💡 Mehrere Frames aus einem Video hinzufügen für einen vielfältigen Trainingsdatensatz.

> ⚠️ **Benötigt: OpenCV**
`pip install opencv-python`
