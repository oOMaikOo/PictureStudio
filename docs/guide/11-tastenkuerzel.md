# ⌨ Tastenkürzel

## Anwendung

| Kürzel | Aktion |
|---|---|
| Strg+N | Neues Projekt |
| Strg+O | Projekt öffnen |
| Strg+S | Projekt speichern |
| Strg+Umschalt+S | Speichern unter… |
| Strg+L | Labels verwalten |
| Strg+K | Kamera aufnehmen |
| Strg+Q | Beenden |
| F1 | Hilfe für aktuelle Seite |

## ROI-Editor (Labeling-Seite)

| Taste | Aktion |
|---|---|
| R | Rechteck-Modus |
| E | Ellipse-Modus |
| G | Polygon-Modus |
| Esc | Zeichnen abbrechen |
| Entf | Ausgewählte ROI löschen |
| Strg+C | ROI kopieren |
| Strg+V | ROI einfügen |
| ↑ ↓ ← → | ROI um 2 px verschieben |
| 1–9 | Label schnell zuweisen |
| N | Nächstes Bild |
| P | Vorheriges Bild |

## Kamera-Dialog

| Taste | Aktion |
|---|---|
| Leertaste | Einzelbild aufnehmen (wenn verbunden) |
